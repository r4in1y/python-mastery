## About

This is a beginner-friendly Python game that simulates the classic **Rock, Paper, Scissors** game.
The computer randomly chooses a move each round, and you try to beat it. The game includes ASCII art for each move to make it fun and visually appealing.

---

## Features

* Simple text-based game
* ASCII art for Rock, Paper, and Scissors
* Input validation to prevent invalid moves
* Option to play multiple rounds

---

## How to Play

1. Run the Python script.
2. Follow the instructions on the screen:

   * Enter `0` for **Rock**
   * Enter `1` for **Paper**
   * Enter `2` for **Scissors**
3. The computer will randomly pick a move.
4. The winner is announced after each round.
5. You can choose to play again or exit.

## Installation

1. Make sure you have **Python 3** installed.
2. Clone the repository:

   ```
   git clone https://github.com/emmaibu/python-mastery/tree/main/003.%20Rock%20Paper%20Scissors
   ```
3. Navigate to the folder:

   ```
   cd Rock Paper Scissors
   ```
4. Run the game:

   ```
   python3 rps.py
   ```
