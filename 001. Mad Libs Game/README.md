# Mad Libs: Adventure in the Amazon Jungle

A wild, interactive Mad Libs game in Python that lets you create your own hilarious adventure stories!
Fill in the blanks with adjectives, nouns, animals, and more, and watch a chaotic jungle adventure unfold. You can also save your story to a file and replay as many times as you want.

## Features
- Iteractive prompts for multiple word types
- Generates a fun, crazy jungle story every time
- Save stories to madlib.txt for later reading
- Play as many times as you like
- Beginner-friendly Python project

## How to Run
1. Clone or download the reopository
2. Run the the Python script

